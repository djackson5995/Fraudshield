# FraudShield

FraudShield is a fraud detection and transaction review platform designed to help businesses identify, investigate, and manage potentially fraudulent transactions before fulfillment.

## Overview

FraudShield combines rule-based risk scoring, transaction monitoring, and case management workflows to support fraud prevention and operational decision-making.

## Core Features

* Transaction risk scoring
* Fraud rule evaluation
* Manual review queue
* Case management and analyst notes
* Reporting and analytics
* Role-based user access

## Technology Stack

* Python
* Flask / FastAPI
* MySQL / PostgreSQL
* HTML, CSS, JavaScript
